
## gdb 命令

| Key binding | Description          |
|-------------|----------------------|
| ~bb~        | set break            |
| ~bc~        | clear break          |
| ~bd~        | delete break         |
| ~bs~        | disable break        |
| ~be~        | enable break         |
| ~c~         | continue             |
| ~i~         | step into            |
| ~J~         | jump to current line |
| ~o~         | step out             |
| ~q~         | quit debug           |
| ~r~         | restart              |
| ~s~         | step over            |
| ~S~         | goto cmd buffer      |
| ~v~         | eval variable        |
